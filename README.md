# API для сайта-портфолио 

Доступно по ссылке [https://ttermechikov.tk/api/**/*](https://ttermechikov.tk/api/projects)

*Для запуска и работы требуется файл **.env**: ```cp .env.example .env```*

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
